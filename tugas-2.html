HyperText Markup Language (HTML) merupakan suatu bahasa markah yang digunakan untuk membuat halaman web, menampilkan berbagai informasi di dalam sebuah browser web. HTML dibuat dengan menggunakan aplikasi pengolah kata dan disimpan dalam format ASCII sehingga menjadi halaman web dengan perintah-perintah HTML. HTML merupakan sebuah standar yang digunakaan secara luas untuk menampilkan halaman web. Pada saat HTML merupakan standar internet yang didefinisikan dan dikendalikan penggunanaannya oleh World Wibe Web Consortium (W3C). HTML dibuat oleh kolaborasi Carillau TIM yang diketuai oleh Berners-lee robert ketika bekerja di CERN pada tahun 1989.
